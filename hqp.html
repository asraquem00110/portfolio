× HOME QUARANTINE PASS MANAGEMENT SYSTEM

DESCRIPTION:

  • A Single Page Application using Vuejs.
  • A Project Idea during enchanced community quarantine because of corona virus.
  • This system manage the generation of Home Quarantine Pass for each family.
  • It checks the validity of that pass through scanning its qrcode.
  • It checks the list of authorized person who can use the pass.
